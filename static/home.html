Architecture

How ThreatGPT Works

01

Input Submission

User pastes suspicious text — email, code snippet, prompt, or any message — into the analyzer.

→
02

LangChain + Groq LLM

Input is passed through a ChatPromptTemplate to llama-3.1-8b-instant via Groq's ultra-fast inference API.

→
03

Resilient JSON Parsing

Output is extracted via 3-layer parser (direct → markdown strip → regex). LLM refusals auto-classify as prompt_injection: critical.

→
04

Structured Report

Results rendered as threat badge, risk gauge, detected pattern chips, and prioritized recommendations.
